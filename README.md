# 🎓 High School Memoir | 高中回忆录网站

> **把那些只存在于记忆里的青春，重新写回时间里。**

**High School Memoir** 是一个轻量级、开源的 Web 回忆录平台，  
致力于帮助高中生 / 毕业生 **记录、保存并分享高中生活的点点滴滴**。

你可以在这里：
- 写下某一次难忘的运动会  
- 记录一次深夜刷题的疲惫  
- 保存一张多年后依然会心一笑的照片  

这是一个 **关于青春、回忆与连接的项目**。

## ✨ 核心功能

- 👤 **用户系统**
  - 注册 / 登录
  - 个人资料管理
  - 支持自定义头像

- 📝 **回忆录发布**
  - 支持文字 + 图片
  - 话题分类（如：高考、运动会、同桌、老师…）

- 💬 **互动系统**
  - 评论、点赞
  - 消息通知（评论 / 点赞提醒）

- 🔥 **话题模块**
  - 创建与浏览热门话题
  - 以主题串联共同记忆

- 📢 **公告系统**
  - 站点公告展示
  - 适合班级 / 学校通知场景

- 📄 **分页与性能优化**
  - 回忆录列表分页
  - 通知分页加载

## 🛠️ 技术栈

> 保持极简、可读、易上手 —— 对新手友好

- **前端**：HTML5 / CSS3 / JavaScript（原生）
- **后端**：PHP（无框架）
- **数据库**：MySQL
- **部署方式**：
  - PHP 内置服务器
  - Nginx / Apache 均可
