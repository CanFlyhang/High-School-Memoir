# High School Memoir - 高中回忆录网站

一个轻量级、功能完整的高中回忆录分享网站，帮助用户记录和分享高中生活的美好时光。

## ✨ 核心功能

- **用户系统**：注册、登录、个人资料管理（支持自定义头像）
- **回忆录发布**：支持文字和图片分享，可选择话题分类
- **互动功能**：评论、点赞、通知系统
- **话题管理**：创建和浏览热门话题
- **公告功能**：网站公告展示
- **分页浏览**：回忆录列表和通知分页显示

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (原生)
- **后端**：PHP (无框架)
- **数据库**：MySQL
- **部署**：支持PHP内置服务器或传统Web服务器

## 📦 快速开始

1. **克隆项目**
   ```bash
   git clone <仓库地址>
   cd High School Memoir
   ```

2. **配置数据库**
   - 创建MySQL数据库
   - 执行 `database.sql` 文件初始化表结构
   - 修改 `api/db.php` 中的数据库连接信息

3. **启动服务**
   ```bash
   php -S localhost:8000
   ```

4. **访问网站**
   打开浏览器访问 `http://localhost:8000`

## 📝 项目特点

- **轻量级**：无复杂框架依赖，易于部署和维护
- **响应式设计**：适配不同屏幕尺寸
- **用户友好**：简洁直观的界面设计
- **功能完整**：满足回忆录分享的核心需求
- **易于扩展**：模块化代码结构，方便添加新功能
