# 📓 High School Memoir · 青春纪念册

**一个轻量级、功能完整的高中回忆录分享网站，帮你珍藏与分享那段闪闪发光的高中时光。**

---

## 🌟 项目简介

你是否还记得同桌的纸条、教室的晚霞、操场上的奔跑？  
「High School Memoir」是一个专为记录与分享高中生活而设计的开源网站项目。你可以在这里写下回忆、上传照片、参与话题、与老同学互动，共同构筑属于你们班级的青春档案馆。

> 🎯 **项目愿景**：让每个人的高中记忆，都有温度可循，有地方可回。

---

## ✨ 核心功能

| 模块 | 功能描述 |
|------|----------|
| 👤 **用户系统** | 注册/登录、个人资料管理、自定义头像 |
| 📝 **回忆录发布** | 支持文字 + 图片混合内容，可关联话题分类 |
| 💬 **互动社区** | 评论、点赞、消息通知系统（实时提醒） |
| 🏷️ **话题管理** | 创建/浏览话题，沉淀班级专属记忆标签 |
| 📢 **公告栏** | 网站重要通知展示，支持管理员发布 |
| 📄 **智能分页** | 回忆录列表、评论、通知均支持流畅分页加载 |

---

## 🛠 技术栈

| 层级 | 技术选型 |
|------|----------|
| **前端** | HTML5 + CSS3 + 原生 JavaScript（无框架依赖） |
| **后端** | PHP（纯手写，无外部框架） |
| **数据库** | MySQL |
| **部署** | 支持 PHP 内置服务器 / Nginx / Apache |

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/CanFlyhang/High-School-Memoir.git
cd High-School-Memoir
```

### 2. 配置数据库
- 创建 MySQL 数据库（如 `school_memoir`）
- 导入 `database.sql` 文件初始化数据表
- 修改 `api/db.php` 中的数据库连接信息

### 3. 启动服务
```bash
php -S localhost:8000
```

### 4. 访问网站
打开浏览器进入：  
👉 **http://localhost:8000**
